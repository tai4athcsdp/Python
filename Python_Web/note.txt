1. Có thể đặt id cho các thẻ html để style chúng 1 cách riêng lẻ.
giá trị id là unique (không trùng lặp)
2. Có thể đặt class cho các thẻ html để style chúng 1 cách group.
giá trị class là not unique (có thể trùng lặp)
3. Khái niệm specificity (độ ưu tiên): trình duyệt sẽ xét giá trị thuộc 
tính nào "mạnh nhất" dựa vào độ ưu tiên
    - inline style
    - id
    - class
    - type
4. Cách dùng thư viện bootstrap
    - vào trang getbootstrap.com, copy <link> tag rồi đưa vào head của file html
    - copy component rồi dùng===> easy
5. Cách dùng biến sass trong html
    - tạo file .scss. Biến trong .scss bắt đầu bằng kí hiệu $
    - cài sass live server trong VSCode, khởi động lại 
    - Kích vào nút "Watch" để biên dịch file .scss ra file .class
    - Dùng thẻ link để link file css vào file html
        + <link ref="style sheet" href="variables.css"> </link>
